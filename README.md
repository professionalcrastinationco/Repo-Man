Toapi is a Python library designed to transform any existing website into a functional REST API, simplifying the process of data extraction. It leverages CSS selectors to define data structures, converting raw HTML output into clean, structured JSON responses. This tool is ideal for developers who need structured access to data from third-party websites lacking official APIs.

**Use Cases:**
- Generating a standardized JSON API from a legacy website or one that only outputs HTML.
- Building data aggregation services by scraping and structuring content from multiple external sources.
- Creating rapid prototypes that require quick access to structured, real-time web data.
